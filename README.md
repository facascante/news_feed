# news_feed
## installation

Pre-requisite
    -- nodejs
    -- npm
    -- mongodb
    --supported OS
```
1. install mongodb on you local or 
    connect to existing mongodb server by modifying the mongourl in config/config.json
2. install npm dependencies
   $> npm install
3. install bower dependencies
   $> bower install
4. Run the application
   $> node bin/www
   
```
